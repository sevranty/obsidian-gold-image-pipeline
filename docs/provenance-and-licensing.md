# Provenance and Licensing Policy

## Code and included source assets

The repository uses the MIT License. The installable skill includes `LICENSE.txt` so the license travels with the direct bundle.

Original programmatic SVG anchors included inside the skill directory are covered by the repository license. OGP#8 raster fixtures are generated working-tree evidence and are not included in the installable bundle.

## Prohibited distribution content

The installable bundle must not contain:

- private design-system documents;
- internal brand assets or third-party logos;
- personal data;
- uploaded user images;
- visual references without recorded rights;
- repository research, reviews, reports, evals, or history;
- hidden caches, editor files, or compiled Python artifacts.

## External inputs

Users remain responsible for rights to references supplied at runtime. A runtime reference is evidence for generation or edit; it does not become part of the distributed skill package.

## Generated outputs

Output licensing depends on the user's source rights, the generation executor terms, and applicable law. The skill does not grant rights in third-party source material or guarantee exclusivity.

## Provenance records

Release evidence records source type, rights status, file path, SHA-256, schema versions, known limitations, and review status. Unknown provenance blocks inclusion.
