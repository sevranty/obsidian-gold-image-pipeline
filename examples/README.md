# OGP#8 Evidence Corpus

This corpus separates three verdict classes:

- `accepted`: five structural examples without declared blocking defects;
- `repairable`: five examples with one localized diagnostic category;
- `rejected`: five examples with critical scope or style violations.

The images are deterministic programmatic QA fixtures, not image-generator outputs. Use them to validate paths, manifests, diagnostics, previews, and regression-runbook behavior. Human visual review remains mandatory.
